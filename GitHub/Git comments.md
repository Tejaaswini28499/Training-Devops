## Git Hub commands:
```
git status
git add .
git commit -m "comments"
git push
git pull- file level
git checkout -b "branch name"- To create a new branch
git checkout "branch name"- switching to new branch
git fetch- repo level
git stash
```

git config --global user.name "Tejaaswini28499"
