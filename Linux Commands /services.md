## Services Linux
```
1. sudo systemctl start <service_name>
2. sudo systemctl stop <service_name>
3. sudo systemctl restart <service_name>
4. sudo systemctl reload <service_name>
5. sudo systemctl enable <service_name>
6. sudo systemctl disable <service_name>
7. sudo systemctl status <service_name>
8. sudo systemctl reboot
9. sudo systemctl is-active
10. sudo systemctl is-enabled
```
